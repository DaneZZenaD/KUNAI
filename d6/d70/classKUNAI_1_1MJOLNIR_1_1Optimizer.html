void KUNAI::MJOLNIR::Optimizer::fallthrough_target_analysis ( MJOLNIR::irgraph_t &  ir_graph)

Analyze the basic blocks of the graph in order to create the fallthrough edges between blocks which are from conditional jumps, these has no edges so for the moment one block looks like goes nowehere: 

              +----------------+
              |                |
              |  jcc           |
              +----------------+

fallthrough / \ target / \ +-------------—+ +-------------—+ | points | | | | nowhere | | jmp | +-------------—+ +-------------—+ | | +-------------—+ | | | | +-------------—+

The one on the left points nowhere but this is the real previous block before the last block, but last block is divided because there's a jump to it, so we will create an edge between, the one on the left, and the last one, as it's a continuation. 

             +----------------+
             |                |
             |  jcc           |
             +----------------+

fallthrough / \ target / \ +-------------—+ +-------------—+ | points | | | | nowhere | | jmp | +-------------—+ +-------------—+ \ / \ / \ / \ / \ / +-------------—+ | | | | +-------------—+
